Mission

We are a RINA Users Group (RUG) in Japan. We are currently working with sponsorship from a communications service provider in Japan. We have weekly meetings to discuss the deployment challenges of RINA. The problems we solve are in preparation for a commercial RINA launch.

Key Performance Indicators

1. Working on solving dynamic namespace management, enrollment, flow allocation, resource allocation, routing, and security issues with a recursive taxonomic namespace service. All RINA stacks publicly available use static namespace management.

2. First commercial RINA deployment in Japan between two datacenters using ethernet / WDM has been accomplished by February 2021.

3. Cardano ADA staking pool, host and relay nodes on the RINA network using IP over RINA has been accomplished by March of 2021.

4. Working with the Cardano community to test and eventually upgrade the IP over RINA to Ouroboros over RINA using ethernet / WDM sans IP.

5. Continue to promote / develop commercially viable RINA solutions and deployments.
